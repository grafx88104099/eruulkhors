## 12. Алдаа гарсан үед

### 12.1 Browser Console-аас бэхээр шалгах
- F12 → Console
- Улаан алдаа байвал бүгдийг screenshot-ож bug report-д хавсарга
- `FirebaseError`, `permission-denied`, `TypeError` гэх мэт тулгуурлах гол үгс

### 12.2 Network tab
- F12 → Network → Filter "firestore", "identitytoolkit", "cloudfunctions"
- 4xx/5xx response-уудыг тэмдэглэ

### 12.3 Hard refresh
- Цөөн bug нь cache-аас үүсдэг. Cmd+Shift+R / Ctrl+Shift+R дарж туршилт давтан хийнэ.
